Classpublic class DiffAreaComputer
ImplementsIDisposable

This class is passed an array of PixelDiff instances and figures out a) how many areas of contiguous pixels the array defines, and b) how large they are. At present we only use this to getLargestDiffAreaDimension() but it would be relatively easy to add getDiffAreaCount(), getDiffAreaRect(index:int), etc.
