Packagecom.allurent.flexunit2.vfu.utils.misc
Classpublic class InputBlocker

InputBlocker prevents user input similar to how the PopUpManager does whenever a modal popup window is created. An invisible pane is drawn over top of the screen which prevents mouseclicks while a FocusManager is created and activated, thereby preventing tabbing to the UI being covered.
