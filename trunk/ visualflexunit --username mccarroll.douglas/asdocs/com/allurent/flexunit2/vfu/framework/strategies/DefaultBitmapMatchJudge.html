Classpublic class DefaultBitmapMatchJudge
ImplementsIBitmapMatchJudge

Default 'BitmapMatchJudge' class used by Visual FlexUnit to judge whether a component's appearance is close enough to the test's saved baseline to pass. The goal is that a result image that isn't visually distinguishable from its baseline should pass, and that one that isn't shouldn't.
