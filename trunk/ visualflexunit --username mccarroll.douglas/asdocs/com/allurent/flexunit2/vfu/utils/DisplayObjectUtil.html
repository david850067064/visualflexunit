center()method
public static function center(displayObject:DisplayObject, container:DisplayObjectContainer):void

Sets the x and y properties on a DisplayObject instance, based on its size and the size and the size of a DisplayObjectContainer. Note that some DisplayObjects, SWFLoader for example, need to load their bitmap before being passed in, otherwise their size will be read as 0 by 0 pixels.

public static function getOutermostDisplayContainer(displayObject:DisplayObject):DisplayObjectContainer

Returns the "outermost" DisplayObjectContainer for a DisplayObject. For example if you pass in a Button contained by a Panel contained by a Canvas, this method would return a reference to the Canvas instance.
